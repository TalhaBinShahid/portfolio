# Talha Bin Shahid — Portfolio

My personal portfolio website built with React, TypeScript, and Tailwind CSS.

## Tech Stack

- **React** + **TypeScript**
- **Vite**
- **Tailwind CSS**
- **shadcn/ui**
- **GSAP** for animations
- **Three.js** for 3D effects

## Getting Started

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
